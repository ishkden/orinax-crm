# Orinax CRM — Design System

> Философия: **Чистота. Воздух. Точность.**  
> Референсы: Linear, Vercel Dashboard, Resend, Notion

---

## Принципы

1. **Минимализм с характером** — никаких лишних элементов, каждый пиксель оправдан
2. **Светлая база** — белый фон, тонкие границы, много воздуха
3. **Один акцентный цвет** — Indigo `#6366F1`, использовать точечно
4. **Анимации — не декор, а ощущение** — всё плавно, ничего не дёргается
5. **Типографика — главный инструмент** — Geist Sans, строгие веса

## Типографика

**Шрифт:** Geist Sans (Vercel) — установлен через пакет `geist`  
**Моноширинный:** Geist Mono — для кодов, ID, технических значений

### Шкала

| Класс       | Размер | Вес  | Использование                    |
|-------------|--------|------|----------------------------------|
| `text-xs`   | 12px   | 400  | Метки, timestamps, badges        |
| `text-sm`   | 14px   | 400  | Основной UI текст                |
| `text-sm`   | 14px   | 500  | Навигация, labels                |
| `text-base` | 16px   | 600  | Заголовки карточек               |
| `text-lg`   | 18px   | 600  | Заголовки секций                 |
| `text-xl`   | 20px   | 700  | Заголовки страниц                |
| `text-2xl`  | 24px   | 700  | Большие числа в метриках         |

**Межстрочный интервал:** `leading-tight` (1.25) для заголовков, `leading-normal` (1.5) для текста  
**Межбуквенный:** `tracking-tight` для заголовков, дефолт для текста

## Анимации (GSAP + Framer Motion)

### Принципы

- **Длительность:** 150–300ms для микроинтеракций, 400–600ms для переходов страниц
- **Easing:** `ease-out` для появления, `ease-in-out` для переходов
- **Никаких bounce** — строгий, профессиональный CRM

### Каталог анимаций

| Анимация                    | Библиотека    | Использование                          |
|-----------------------------|---------------|----------------------------------------|
| Hover на карточках          | CSS Tailwind  | `hover:shadow-md transition-shadow`    |
| Появление страницы          | Framer Motion | Fade + translateY(8px → 0)             |
| Счётчик метрик              | GSAP          | Числа считаются от 0 до значения       |
| Sidebar активный пункт      | Framer Motion | Layout animation на индикаторе         |
| Toast/уведомления           | Framer Motion | Slide in from right                    |
| Таблицы — появление строк   | Framer Motion | Stagger children                       |

### Пример счётчика (GSAP)
```tsx
// Числа в метриках анимированно считаются при монтировании
gsap.from(counterRef.current, {
  textContent: 0,
  duration: 1.2,
  ease: "power2.out",
  snap: { textContent: 1 },
});
```

### Пример перехода страниц (Framer Motion)
```tsx
// Обёртка для каждой страницы
<motion.div
  initial={{ opacity: 0, y: 8 }}
  animate={{ opacity: 1, y: 0 }}
  transition={{ duration: 0.3, ease: "easeOut" }}
>
  {children}
</motion.div>
```
